## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   bun install
   ```

3. Set up the database:
   ```bash
   bun run db:push
   ```

4. Start the development server:
   ```bash
   bun run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to view the application.
